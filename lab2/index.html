Tags and Styles Used

HTML TAGS DESCRIPTION
Div tags To create Specific sections for each content.
UL and LI To list the items in on the page.
Anchor Tag To provide links to specific location or sites .
Heading & paragraph tags To incorporate Text on the page
Underline & Italics tag To underline or make the words italic.
BR tag To incorporate the next line .
Table tags To create atables and add description.
CSS STYLES DESCRIPTION
Background Color To give background Colour
Text Alignment To align the text in a specific position.
Color To add color to text.
Margin and Padding To add space between each section and contents within that section.
Link , Visited , Hover To add effects before , on hover , and after visiting the links.
Height and Width To adjust size of content .
Image Radius To make the image border round .
